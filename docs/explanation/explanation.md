# Explanation

Background reading. These pages cover the *why* — the underlying methodology, anatomy and design rationale — rather than how to use a specific command.

- [Diffusion MRI](diffusion-mri.md) — DWI contrast, b-value, the diffusion tensor and the four DTI scalar metrics csttool reports.
- [Tractography](tractography.md) — deterministic vs probabilistic propagation, the CSA-ODF model used by csttool, and atlas-based ROI filtering.
- [CST Anatomy](cst-anatomy.md) — the corticospinal tract trajectory from motor cortex to spinal cord, key landmarks, and ALS-related microstructural change.
- [Design Decisions](design-decisions.md) — choices baked into csttool (e.g. bilateral extraction, bidirectional seeding).
- [Known Limitations](limitations.md) — what csttool can and cannot do.
- [References](references.md) — bibliography for the scholarly citations on these pages.
